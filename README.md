# multilabel_classification_of_laws

Предобработка .xml файлов. Перебор различных векторизаторов, включая **transformers**. Подбор наилучшей модели, подбор наиболее подходящих гиперпараметров. Подбор оптимальной размерности векторов с помощью метода главных компонент. Создание и обучение нейронной сети для задачи **multilabel_classification_of_laws**, векторизация дополнительных признаков, таких как **"Organ"**,**"Name"**, основной признак **"Text"**, использование байесовского оптимизатора для подбора гиперпараметров. Проверка выведенных меток и решение проблемы несбалансированности классов.
